🚀 快速启动

三步启动完整系统

01

启动基础设施

docker-compose -f docker/docker-compose.yml up -d

一键启动 MySQL、Redis、RabbitMQ、Nacos 等基础服务

02

初始化数据库

mysql -h localhost -P 3307 -u root -proot123 < smart-interview-common/src/main/resources/sql/init.sql

执行 SQL 脚本初始化数据结构

03

启动微服务

Gateway → User → Interview → AI

依次启动各微服务模块，即可通过 Gateway 8080 端口访问
